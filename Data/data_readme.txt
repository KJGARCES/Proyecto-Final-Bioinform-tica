Las secuencias utilizadas en este análisis filogenético corresponden al gen ribosomal 12S rRNA y fueron obtenidas del repositorio GenBank. En total, se incluyeron 20 secuencias pertenecientes a diversas especies del género Pristimantis y una secuencia del género Oreobates utilizada como grupo externo. Las especies de Pristimantis incluidas son: Pristimantis orestes, Pristimantis bambu, Pristimantis galdi, Pristimantis kichwarum, Pristimantis lacrimosus, Pristimantis lanthanites, Pristimantis librarius, Pristimantis luscombei, Pristimantis malkini, Pristimantis martiae, y Pristimantis matidiktyo. Las secuencias varían en longitud pero fueron alineadas para obtener una longitud uniforme de 775 nucleótidos por secuencia. Este alineamiento preciso es crucial para asegurar la exactitud de los análisis filogenéticos subsiguientes, proporcionando una base sólida para la inferencia de las relaciones evolutivas dentro del género Pristimantis.
